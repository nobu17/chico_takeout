# Chico SP Takeout

Takeout api and web site of Chico Spice.


# Server side
Go, Gin

## test db (docker)

### launch
cd ./test_env
docker-compose up -d

### stop
cd ./test_env
docker-compose stop


## how to run
```sh
go run main.go
```


# Frontend
React

## how to run
```sh
cd frontend
yarn start
```
